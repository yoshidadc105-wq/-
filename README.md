# 社内マニュアルシステム

Teachme Biz からデータを移行した社内マニュアル管理システムです。

## 機能

- PDFマニュアルのアップロード・閲覧
- リッチテキストマニュアルの作成・編集
- カテゴリ分類・検索
- ユーザー管理（人数制限なし）
- 閲覧履歴
- 管理者ダッシュボード

## セットアップ

### 必要なもの
- Node.js 18以上

### インストール

```bash
npm install
```

### 起動

```bash
npm start
```

ブラウザで http://localhost:3000 を開く。

### 初期ログイン

| 項目 | 値 |
|------|-----|
| ユーザー名 | admin |
| パスワード | admin123 |

**⚠️ ログイン後すぐにパスワードを変更してください。**

## ディレクトリ構成

```
├── server.js          # サーバーエントリーポイント
├── src/
│   ├── db.js          # データベース初期化
│   ├── routes/        # APIルート
│   └── middleware/    # 認証ミドルウェア
├── public/            # フロントエンド
│   ├── css/
│   ├── js/
│   ├── index.html     # マニュアル一覧
│   ├── view.html      # マニュアル閲覧
│   ├── create.html    # マニュアル作成・編集
│   └── admin.html     # 管理者設定
├── uploads/           # PDFファイル保存先（自動生成）
└── data/              # SQLiteデータベース（自動生成）
```

## PDFのインポート方法

1. ログイン後、「新規マニュアル作成」をクリック
2. 「PDFをアップロード」タブを選択
3. Teachme BizからエクスポートしたPDFを選択してアップロード

## ホスティング

ローカルで動かす場合はそのまま利用できます。  
クラウドで公開する場合は Railway / Render などに対応しています。
